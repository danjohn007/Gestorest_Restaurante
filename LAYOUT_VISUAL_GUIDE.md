## Flujo de Trabajo: Crear Pedido desde Layout

### Antes (Flujo Antiguo)
```
Usuario โ Menรบ Pedidos โ Crear Pedido โ Seleccionar Mesa โ Seleccionar Platillos
         (3 clics)                     (buscar en dropdown)
```

### Ahora (Flujo Mejorado)
```
Usuario โ Layout de Mesas โ Hover sobre Mesa โ Clic "Nuevo Pedido" โ Platillos
         (2 clics)                              (mesa preseleccionada!)
```

**Ahorro: 1 clic menos + Mesa automรกticamente seleccionada**
